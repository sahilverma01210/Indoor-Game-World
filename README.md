* Chess
* Carrom
* Ludo
* Snake & Ladder

# Package &amp; Library Dependencies
* Universal RP
* ARCore XR Plugin
* AR Foundation
* XR Plugin Management
* NuGet
* Web Socket Sharp (from NuGet Gallery)
* Newtonsoft JSON
* JSONSerialize
* Node JS

# Other Package Dependencies
* Particle System
* Text Mesh Pro
* Unity UI
* Android Logcat
* Screen Capture
* Terrain
