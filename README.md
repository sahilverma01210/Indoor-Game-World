# Indoor-Game-World
Casual 3D &amp; 3D Augmented Reality Multiplayer Game made with Unity using Universal Render Pipeline (URP) for Android.

# Packages Used
* Universal RP
* ARCore XR Plugin
* AR Foundation
* XR Plugin Management
* NuGet
* Web Socket Sharp (from NuGet Gallery)
* Newtonsoft JSON
* JSONSerialize

# Other Packages Used
* Particle System
* Text Mesh Pro
* Unity UI
* Android Logcat
* Screen Capture
* Terrain
